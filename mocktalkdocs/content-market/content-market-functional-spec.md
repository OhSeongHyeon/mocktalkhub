# 콘텐츠 / 환율·금 시세 기능정의서

## 문서 목적

- `콘텐츠` 메뉴와 하위 화면의 현재 동작을 사용자 관점에서 정의한다.
- `이미지 갤러리(준비중)`, `환율/금 시세`, 백오피스 운영 화면의 상태별 동작을 정리한다.
- 공개 조회와 관리자 운영 기능의 차이를 실제 UI/API 기준으로 명확히 남긴다.

## 1. 기능 개요

사이드 메뉴의 `콘텐츠`는 아래 두 콘텐츠를 묶는 허브 진입점이다.

- `이미지 갤러리`
- `환율/금 시세`

현재 실제 사용 기능은 `환율/금 시세`이며 `이미지 갤러리`는 준비중 카드로만 노출한다.

## 2. 대상 사용자

### 2.1 공개 사용자

- 비로그인 사용자
- 로그인 사용자

`/contents`, `/contents/market`는 로그인 여부와 무관하게 동일하게 접근 가능하다.

### 2.2 운영 사용자

- `ADMIN`
- `MANAGER`

운영자는 `/admin/content-market`에서 최신화와 데이터 임포트를 수행할 수 있다.

## 3. 화면 정의

### 3.1 사이드 메뉴

- 메뉴명: `콘텐츠`
- 경로: `/contents`
- 아이콘: 기존 `gallery` 아이콘 재사용

### 3.2 콘텐츠 허브 화면

허브 화면은 2개 카드로 구성된다.

#### 3.2.1 이미지 갤러리 카드

- 제목: `이미지 갤러리`
- 상태: `준비중`
- 동작: 링크 없이 준비중 안내만 노출

#### 3.2.2 환율/금 시세 카드

- 제목: `환율/금 시세`
- 상태: 사용 가능
- 동작: `/contents/market`으로 이동

### 3.3 환율/금 시세 상세 화면

상세 화면은 아래 블록으로 구성된다.

- 상단 설명 영역
  - 마지막 갱신 시각
  - `새로고침` 버튼
- 기간 선택 영역
  - `10년`
  - `5년`
  - `3년`
  - `1년`
  - `6개월`
  - `3개월`
  - `30일`
  - `7일`
  - `시작일 / 종료일` 직접 선택
- 통합 그래프 영역
  - 전체 종목 멀티 시리즈 차트
  - 기준일 값을 100으로 맞춘 상대지수 y축
- 종목 선택 영역
  - `USD/KRW`
  - `EUR/KRW`
  - `JPY/KRW`
  - `금 시세 (USD/g)`
  - `금 시세 (KRW/g)`
- 선택 종목 상세 영역
  - 현재 값
  - 변동
  - 단일 시리즈 그래프
  - 평균값
  - 중위값
  - 최저값
  - 최고값

### 3.4 관리자 운영 화면

관리자 화면은 아래 두 블록으로 구성된다.

- `지금 최신화`
- `시세 데이터 임포트`

임포트 화면은 `통합 파일`, `종목별 파일` 두 모드를 제공한다.

## 4. 제공 데이터 범위

### 4.1 환율

- `USD/KRW`
- `EUR/KRW`
- `JPY/KRW`

### 4.2 금 시세

- `XAU/USD` 원천값
- `XAU/KRW` 파생값

공개 화면에서는 두 금 시세를 모두 `/g` 기준으로 환산해 노출한다.

## 5. 사용자 동작 정의

### 5.1 콘텐츠 허브 진입

- 사용자가 `콘텐츠` 메뉴를 클릭한다.
- 시스템은 허브 화면을 노출한다.

### 5.2 환율/금 시세 상세 진입

- 사용자가 `환율/금 시세` 카드를 클릭한다.
- 시스템은 `/contents/market`으로 이동한다.
- 시스템은 `overview` 1회 조회 후 기본 기간 `YEAR` 기준 series를 종목별로 병렬 조회한다.

### 5.3 기간 전환

- 사용자가 프리셋 기간 버튼을 누른다.
- 시스템은 선택 기간으로 전체 종목 시계열을 다시 조회한다.
- 통합 그래프와 선택 종목 상세 그래프는 같은 기간으로 동기화된다.

### 5.4 직접 기간 선택

- 사용자가 `시작일`, `종료일`을 입력하고 `직접 선택 적용`을 누른다.
- 시스템은 `period=CUSTOM`으로 시계열을 다시 조회한다.
- 시작일 또는 종료일이 비어 있으면 범위 에러를 노출한다.
- 시작일이 종료일보다 늦으면 범위 에러를 노출한다.

### 5.5 종목 전환

- 사용자가 종목 선택 카드를 누른다.
- 시스템은 이미 불러온 시계열 중 선택 종목의 현재 값, 변동, 통계, 상세 그래프를 보여준다.

### 5.6 새로고침

- 사용자가 상단 `새로고침` 버튼을 누른다.
- 시스템은 공개 API `overview`, `series`를 다시 조회한다.
- 이 버튼은 관리자 최신화 API를 호출하지 않는다.

### 5.7 자동 갱신

- 사용자가 상세 화면에 머무는 동안 30분마다 공개 API 재조회가 실행된다.
- 사용자가 화면을 벗어나면 폴링은 중단된다.
- 백엔드 원천 수집은 하루 1회가 기본이다.

## 6. 상태별 동작

### 6.1 로딩 상태

- overview 조회 중: `시세 요약 정보를 불러오는 중입니다.`
- series 조회 중: `선택 범위 + 통합 그래프를 불러오는 중입니다.`

### 6.2 빈 상태

- overview 데이터가 없으면 `아직 집계된 시세 데이터가 없습니다. 배포 후 첫 수집이 완료되면 그래프를 확인할 수 있습니다.`를 노출한다.
- 상세 시계열이 부족하면 `표시할 수 있는 시계열 데이터가 아직 충분히 쌓이지 않았습니다.` 또는 `선택한 기간의 시세 데이터가 아직 충분히 쌓이지 않았습니다.`를 노출한다.
- 마지막 갱신 시각이 없으면 `데이터를 축적하는 중입니다.`를 표시한다.

### 6.3 에러 상태

- overview 조회 실패 시 요약 에러 문구를 노출한다.
- series 전체 실패 시 `시계열 데이터를 불러오지 못했습니다.`를 노출한다.
- 일부 종목만 실패하면 `일부 종목의 시계열 데이터를 불러오지 못했습니다.`를 노출한다.
- 직접 선택 기간 오류는 별도 범위 에러 문구로 노출한다.

### 6.4 준비중 상태

- `이미지 갤러리` 카드는 항상 준비중 상태를 유지한다.
- 사용자는 허브에서 실제 제공 콘텐츠와 준비중 콘텐츠를 구분할 수 있어야 한다.

## 7. API 기능 정의

### 7.1 환율/금 시세 요약 API

- 엔드포인트: `GET /api/contents/market/overview`
- 인증: 불필요
- 역할: 종목별 최신값, 직전 대비 값, 전체 마지막 갱신 시각 제공

### 7.2 환율/금 시세 시계열 API

- 엔드포인트: `GET /api/contents/market/series`
- 인증: 불필요
- 필수 파라미터:
  - `instrument`
  - `period`
- `period=CUSTOM`일 때 추가 파라미터:
  - `startDate`
  - `endDate`

예시:

- `GET /api/contents/market/series?instrument=USD_KRW&period=YEAR`
- `GET /api/contents/market/series?instrument=XAU_USD&period=TEN_YEAR`
- `GET /api/contents/market/series?instrument=USD_KRW&period=CUSTOM&startDate=2026-01-01&endDate=2026-03-15`

### 7.3 백오피스 최신화 API

- 엔드포인트: `POST /api/admin/contents/market/refresh`
- 인증: `ADMIN`, `MANAGER`
- 역할: 현재 시세를 즉시 1회 다시 수집

### 7.4 백오피스 임포트 API

- 엔드포인트: `POST /api/admin/contents/market/import`
- 인증: `ADMIN`, `MANAGER`
- 입력:
  - multipart `file`
  - 선택 query param `instrument`
- 역할: 과거 시세 데이터를 CSV/XLSX 파일로 업로드

## 8. 갱신 방식 정의

- 백엔드
  - 외부 시세를 하루 1회 수집
  - 앱 시작 시 1회 수집 옵션 지원
  - 운영자는 수동 최신화를 실행 가능
- 프론트
  - 상세 화면 진입 시 즉시 조회
  - 상세 화면 유지 중 30분 폴링
  - `새로고침`은 공개 API 재조회
- 과거 데이터 보강
  - 백오피스 `CSV/XLSX 임포트`
- 장기 기간 조회
  - `1년`, `3년`, `5년`, `10년`은 백엔드 Redis 캐시 활용

`SSE`, `WebSocket`은 현재 범위에 포함하지 않는다.

## 9. 접근 정책

- `/contents`, `/contents/market`: 공개 읽기
- `/admin/content-market`: 운영 권한 필요
- 관리자 API: `ADMIN`, `MANAGER` 전용

## 10. 현재 범위

- `콘텐츠` 메뉴와 허브 화면
- `이미지 갤러리(준비중)` 카드
- `환율/금 시세` 상세 화면
- 공개 overview / series API
- `10년 / 5년 / 3년 / 1년 / 6개월 / 3개월 / 30일 / 7일 / 직접 선택`
- 통합 그래프 상대지수 비교
- 선택 종목 통계
- 하루 1회 자동 수집
- 앱 시작 시 1회 수집 옵션
- 관리자 `지금 최신화`
- 관리자 `CSV/XLSX 임포트`
- 장기 preset 기간 Redis 캐시

현재 범위에 포함하지 않는 항목:

- 외부 API 기간 재조회 방식 백필
- 실시간 스트리밍
- 이미지 갤러리 실제 기능

## 11. 후속 고도화 후보

- 이미지 갤러리 실제 기능 구현
- 추가 시세 종목 확대
- 임포트 작업 이력 화면
- 임포트 템플릿 다운로드
- 수집 상태 모니터링
